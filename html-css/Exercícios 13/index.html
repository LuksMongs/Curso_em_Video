Estilos Locais ou Internos

Para realizar as alterações do design (estilo) em partes do texto sem utilizar o INLINE.
- Na área HEAD crie uma TAG STYLE
- Para cada área a ser alterada deve-se criar um SELETOR, nesse caso iniciaremos com um seletor pro BODY

- As TAGs ficam entre < cochete angular > e o SELETOR é adicionado na HEAD e seu conteúdo fica entre { chaves }.
