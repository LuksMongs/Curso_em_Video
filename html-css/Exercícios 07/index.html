Marcação de Texto

Podemos criar também um trecho ou texto marcado usando a tag MARK

Podemos usar .CSS dentro do html para conseguir um detalhe pontual ou específico como mudar a cor da marcação colocando
o parâmetro STYLE dentro da tag MARK alterando o BACKGROUND.
o CSS desse caso foi (MARK STYLE="BACKGROUN-COLOR: RGB(255, 0, 170);

Criar um SELETOR

no compartimento HEAD, podemos colocar uma tag com SELETOR {} para realizar uma alteração em todas as tags, com mesmo nome, de uma só vez.
Por exemplo o MARK. Criando na HEAD uma tag STYLE e SELETANDO o MARK conseguimos mudar a cor dos marcadores por exemplo.
Veja a mágica acontecer. Vá na HEAD e mude a cor do MARK lá. Outra maneira é usando link externo (arquivo .css linkado ao html)
