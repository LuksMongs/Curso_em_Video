Código-Fonte / Pré-Formatação

São comandos ou parte de códigos. por exemplo o comando document.gentlementById('teste') é escrito em linguagem JavaScript.
Nesse exemplo acima ler o comando é embolado por ser uma fonte não MONOESPAÇADA. Para torna-la uma fonte do tipo
MONOESPAÇADA eu terei que usar o CODE. Ficando document.gentlementById('teste'). Os espaços de cada letra são iguais entre eles.

Contudo o CODE não respeita quebra de linha. segue o exemplo abaixo:
Usando apenas CODE:
num = int(input('Digite um númeo')) if num % 2 == 0: print(f'O número {num} é PAR) else: print(f'O número é IMPAR) print('Fim do Programa')

Usando PRE e CODE: 

                
            num = int(input('Digite um númeo'))
            if num % 2 == 0:
                print(f'O número {num} é PAR)
            else:
                print(f'O número é IMPAR)
            print('Fim do Programa')
