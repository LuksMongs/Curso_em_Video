Usando Links Externos

Você pode acessar um local da web através de um link específico como esse por exemplo que foi adicionado ao código html.
utilizando a ANCORA < A > e o parâmetro de referência HREF="enderco_http_do_local_da_web".
dessa maneira o direcionamento é feito na mesma aba do navegador.

Acessar um endereço com a abertura de uma NOVA aba no navegador, é feita adicionando o parâmetro TARGET="_BLANK" ao comando <a href="http://youtube.com/" targe="_blank">

Hoje é indicado ADICIONAR além do TARGET o parâmetro REL=" ". ficando assim:
<a href="http://youtube.com/" targe="_blank" rel="extenal">. Testando: Youtube

Usando Links Internos

Para criar um link interno, ou seja, entre as páginas criadas em um mesmo local também utilizamos a ANCORA (A) com parâmetro (href=" "). Contudo a indicação do HREF será
um caminho local e não um HYPERLINK (endereço web). E o complemento é o parâmetro REL="PREV ou NEXT".
Para visitar a minha Págia 02, clique no link Página 02.

Para avisar ao mecanismo de busca que não há um aval acerca do link que você está disponibilizando,
adicione ao REL="nofollow". É um aviso sobre isso (não entendi nada desse NOFOLLOW). Segue o exemplo:
