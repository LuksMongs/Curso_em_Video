Listas Ordenadas

São listas que a ordem é IMPORTANTE. Para não escrever a lista toda colocando os números um a um, segue ex.
1- acordar
2- levantar
3- escovar os dente
4- pentear o cabelo
5- arrumar a cama...
em uma lista assim, caso queira adicinar um item, terá que alterar a numeração toda manualmente.

Portando devemos usar o comando (tag) OL (lista ordenada - numerada) e UL (lista nao-ordenada - somente tabulaçao).
Cada item dessas listas devem estar marcados (entre) com a tag LI. ex.

  1. Acordar
  2. Levantar
  3. Escovar os dente
  4. Penterar o cabelo
  5. Arrumar a cama
  6. ...

    7. essa lista pode ser personalizada usando o parâmetro TYPE ficando <ol type=" " > e pode iniciar a contagem a
    partir do número escolhido, com o parâmetro START ficando <ol start=" nº escolhido">. ou colocando tudo junto: <ol type=" " start=" nº escolhido">.
    O type tem 4 opções: 1. (numérico) A.(ordem alfabética Maiúscula) a.(alfab. minuscula) I.(romanos Maiúscula) i.(romano minuscula)
    E na lista NÃO oredenada tem as opções: type="disco/circle/square".

Para uma lista NÃO ordenada (UL) segue o exemplo.

    Preciso Comprar:
  • pão
  • Leite
  • Escovar de dentes
  • queijo
  • manteiga
  • ...

Uma LISTA pode conter outras (sublistas) usando os mesmos princípios de OL ou UL dentro de uma LI.

  • Misto Quente
    1. pão
    2. presunto
    3. queijo
    4. manteiga
  • Hamburger
    • pao

      2. com parmesão
      3. integral
    • carne

e uma DESCRIÇÃO (tag DD) segue exemplo.

CSS
