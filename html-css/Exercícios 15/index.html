Capítulo 1.1

Estilos a partir de CSS Externo

-Criar um arquivo .cc (normalmente com nome STYLE.CSS na mesma pasta do INDEX
- Iniciar com a tag CHARSET="UTF-8"
- Montar o código a para as alterações de DESIGN.
- No CSS o MARCADOR é iniciado pelo nome da área que deseja modificar (por exemplo: Body { }, h1{ }, h2{ }, p{ }...)

É possível também mesclar (usar ao mesmo tempo em um código HTML) o STYLE INLINE > O STYLE NA HEAD > o CSS EXTERNO Seguindo essa ordem de predominância.
